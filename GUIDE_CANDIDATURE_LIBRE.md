# Guide d'utilisation - Candidature Libre

## Vue d'ensemble

La fonctionnalité de **Candidature Libre** permet aux enseignants de proposer des cours non répertoriés dans le système avec leur équipe d'enseignement complète, incluant tous les volumes horaires détaillés.

## Accès à la fonctionnalité

### Depuis la page d'accueil
1. Cliquez sur le bouton **"Actions"** dans le menu déroulant
2. Sélectionnez **"Candidature libre"** dans les options disponibles

### Depuis le header
- Cliquez directement sur le bouton **"Candidature libre"** dans l'en-tête de la page

## Processus de candidature

### 1. Informations du soumissionnaire
- **Nom complet** : Votre nom et prénom (obligatoire)
- **Email** : Votre adresse email institutionnelle (obligatoire)
- **Téléphone** : Votre numéro de téléphone (optionnel)

### 2. Informations du cours
- **Code du cours** : Code unique du cours (ex: LINFO1001) (obligatoire)
- **Faculté** : Sélectionnez la faculté concernée (obligatoire)
- **Nom du cours (français)** : Titre du cours en français (obligatoire)
- **Nom du cours (anglais)** : Titre du cours en anglais (optionnel)
- **Sous-catégorie** : Sous-catégorie spécifique (optionnel)
- **Volume horaire Vol.1** : Nombre d'heures pour le volume 1 (obligatoire)
- **Volume horaire Vol.2** : Nombre d'heures pour le volume 2 (obligatoire)
- **Description du cours** : Description détaillée du contenu (optionnel)
- **Prérequis** : Conditions préalables pour suivre le cours (optionnel)
- **Objectifs d'apprentissage** : Objectifs pédagogiques (optionnel)

### 3. Coordonnateur du cours
- **Nom et Prénom** : Nom complet du coordonnateur (obligatoire)
- **Entité** : Institut, département ou unité (obligatoire)
- **Statut** : Statut académique (Professeur, Chargé de cours, etc.) (obligatoire)
- **Email** : Adresse email du coordonnateur (optionnel)
- **Téléphone** : Numéro de téléphone (optionnel)
- **Volume Vol.1** : Heures d'enseignement du coordonnateur pour Vol.1 (obligatoire)
- **Volume Vol.2** : Heures d'enseignement du coordonnateur pour Vol.2 (obligatoire)

### 4. Cotitulaires
- **Ajout de cotitulaires** : Cliquez sur "Ajouter un cotitulaire" pour chaque membre supplémentaire
- **Informations requises** : Mêmes champs que pour le coordonnateur
- **Suppression** : Utilisez le bouton "Supprimer ce cotitulaire" si nécessaire

## Validation des volumes horaires

### Contrôles automatiques
- Le système vérifie automatiquement que la somme des volumes proposés correspond aux volumes requis
- **Volume 1** : Total des heures Vol.1 du coordonnateur + cotitulaires = Volume total Vol.1 du cours
- **Volume 2** : Total des heures Vol.2 du coordonnateur + cotitulaires = Volume total Vol.2 du cours

### Indicateurs visuels
- ✅ **Vert** : Volumes correspondants
- ❌ **Rouge** : Volumes ne correspondant pas
- **Détails** : Affichage des écarts en heures

## Statuts académiques disponibles

- **PROF** : Professeur
- **CHARG** : Chargé de cours
- **MAITRE** : Maître de conférences
- **ASSIST** : Assistant
- **DOCT** : Docteur
- **AUTRE** : Autre statut

## Facultés disponibles

- **FSM** : Faculté des Sciences de la Motricité
- **FSP** : Faculté de Santé Publique
- **FASB** : Faculté d'Architecture et d'Urbanisme
- **MEDE** : Faculté de Médecine et Médecine Dentaire

## Soumission de la candidature

### Conditions de validation
1. Tous les champs obligatoires sont remplis
2. Les volumes horaires correspondent exactement
3. Au moins un coordonnateur est défini

### Processus de soumission
1. Vérifiez que tous les indicateurs sont verts
2. Cliquez sur **"Soumettre la candidature"**
3. Confirmation de soumission avec message de succès
4. Redirection automatique vers la page d'accueil

## Suivi de la candidature

### Statuts possibles
- **En attente** : Candidature soumise, en cours d'examen
- **Approuvée** : Candidature acceptée par l'administration
- **Rejetée** : Candidature refusée avec justifications

### Notifications
- Les notifications de statut sont envoyées par email
- Le suivi peut être consulté dans l'interface d'administration

## Conseils d'utilisation

### Avant la soumission
- Vérifiez attentivement tous les volumes horaires
- Assurez-vous que les informations de contact sont correctes
- Rédigez une description claire du cours

### Gestion des volumes
- Commencez par définir les volumes totaux du cours
- Répartissez ensuite les heures entre les membres de l'équipe
- Utilisez les indicateurs visuels pour vérifier la cohérence

### Équipe d'enseignement
- Définissez clairement le rôle de coordonnateur
- Ajoutez tous les cotitulaires nécessaires
- Vérifiez les statuts académiques de chaque membre

## Support et assistance

En cas de difficulté ou de question :
- Consultez ce guide d'utilisation
- Contactez l'équipe d'administration
- Vérifiez que votre navigateur est à jour

---

**Note** : Ce système est conçu pour faciliter la gestion des candidatures libres tout en maintenant la cohérence des données. Toute candidature soumise sera examinée par l'administration avant validation finale.
