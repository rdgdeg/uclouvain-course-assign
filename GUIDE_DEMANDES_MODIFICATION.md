# Guide d'utilisation - Demandes de Modification

## Vue d'ensemble

La fonctionnalité de **Demandes de Modification** permet aux utilisateurs de consulter tous les cours avec leurs enseignants et volumes horaires, et de soumettre des demandes de modification pour des cours existants ou non répertoriés.

## Accès à la fonctionnalité

### Depuis la page d'accueil
1. Cliquez sur le bouton **"Actions"** dans le menu déroulant
2. Sélectionnez **"Demander une modification"** dans les options disponibles

### Depuis le header
- Cliquez directement sur le bouton **"Demandes modification"** dans l'en-tête de la page

## Consultation des cours

### Affichage des cours
La page affiche tous les cours disponibles avec les informations suivantes :
- **Titre et code** du cours
- **Faculté** et sous-catégorie
- **Volumes horaires** (Vol.1 et Vol.2)
- **Statut** : Vacant ou Attribué
- **Équipe d'enseignement** avec les détails de chaque enseignant

### Filtrage et recherche
- **Barre de recherche** : Recherchez par code ou titre de cours
- **Filtre par faculté** : Filtrez les cours par faculté spécifique
- **Bouton "Cours non répertorié"** : Pour les cours qui ne sont pas dans la liste

## Types de demandes

### 1. Modification d'un cours existant
- Sélectionnez un cours dans la liste
- Cliquez sur **"Demander modification"**
- Remplissez le formulaire de demande

### 2. Modification d'un cours non répertorié
- Cliquez sur **"Cours non répertorié"**
- Saisissez les informations du cours concerné
- Remplissez le formulaire de demande

## Processus de demande

### Informations requises

#### Pour un cours existant
- **Cours concerné** : Pré-rempli avec les informations du cours sélectionné
- **Nom complet** : Votre nom et prénom (obligatoire)
- **Email** : Votre adresse email institutionnelle (obligatoire)
- **Type de modification** : Sélectionnez le type approprié (obligatoire)
- **Description détaillée** : Décrivez la modification demandée (obligatoire)

#### Pour un cours non répertorié
- **Informations du cours** :
  - Code du cours
  - Faculté
  - Titre du cours
  - Description du cours
- **Informations du demandeur** : Mêmes champs que pour un cours existant
- **Type de modification** et **Description détaillée**

### Types de modification disponibles

- **Attribution d'enseignant** : Changement dans l'équipe d'enseignement
- **Modification du contenu** : Changement dans le contenu du cours
- **Modification des horaires** : Changement dans les horaires
- **Modification des volumes horaires** : Changement dans les volumes
- **Autre** : Autres types de modifications

## Validation et soumission

### Contrôles automatiques
- Vérification que tous les champs obligatoires sont remplis
- Validation du format email
- Contrôle de la cohérence des informations

### Processus de soumission
1. Remplissez tous les champs obligatoires
2. Vérifiez la description de votre demande
3. Cliquez sur **"Envoyer la demande"**
4. Confirmation de soumission avec message de succès

## Suivi des demandes

### Statuts possibles
- **En attente** : Demande soumise, en cours d'examen
- **En cours de traitement** : Demande en cours d'analyse
- **Approuvée** : Demande acceptée et appliquée
- **Rejetée** : Demande refusée avec justifications

### Notifications
- Les notifications de statut sont envoyées par email
- Le suivi peut être consulté dans l'interface d'administration

## Informations affichées pour chaque cours

### Détails du cours
- **Titre** : Nom complet du cours
- **Code** : Code unique du cours
- **Faculté** : Faculté responsable
- **Sous-catégorie** : Sous-catégorie spécifique (si applicable)

### Volumes horaires
- **Volume 1** : Nombre d'heures pour le premier volume
- **Volume 2** : Nombre d'heures pour le deuxième volume

### Statut du cours
- **Vacant** : Aucun enseignant assigné
- **Attribué** : Au moins un enseignant assigné

### Équipe d'enseignement
Pour chaque enseignant assigné :
- **Nom** : Nom complet de l'enseignant
- **Statut** : Statut académique (Professeur, Chargé de cours, etc.)
- **Rôle** : Coordonnateur ou cotitulaire
- **Volumes** : Heures d'enseignement pour Vol.1 et Vol.2

## Conseils d'utilisation

### Avant de soumettre une demande
- Vérifiez que le cours n'est pas déjà dans la liste
- Assurez-vous d'avoir toutes les informations nécessaires
- Rédigez une description claire et détaillée

### Rédaction de la description
- Soyez précis sur la modification demandée
- Justifiez la demande avec des arguments pertinents
- Incluez toute information contextuelle utile
- Mentionnez les impacts potentiels

### Gestion des cours non répertoriés
- Fournissez autant d'informations que possible sur le cours
- Indiquez clairement pourquoi le cours n'est pas dans la liste
- Précisez le contexte de la demande

## Support et assistance

En cas de difficulté ou de question :
- Consultez ce guide d'utilisation
- Contactez l'équipe d'administration
- Vérifiez que votre navigateur est à jour

## Exemples de demandes

### Exemple 1 : Modification d'attribution
**Type** : Attribution d'enseignant
**Description** : "Le professeur Dupont ne peut plus assurer le cours LINFO1001 pour des raisons de santé. Je propose que le Dr Martin, qui a déjà enseigné ce cours l'année dernière, le remplace. Dr Martin est disponible et a l'expertise nécessaire."

### Exemple 2 : Modification de contenu
**Type** : Modification du contenu
**Description** : "Je souhaite ajouter un module sur les nouvelles technologies web (React, Vue.js) au cours LINFO2001. Ce module représenterait 10h supplémentaires et serait enseigné par le Dr Garcia qui a une expertise dans ce domaine."

### Exemple 3 : Cours non répertorié
**Type** : Autre
**Description** : "Je souhaite proposer un nouveau cours 'Introduction à l'Intelligence Artificielle' (code: LINFO3001) pour les étudiants de 3ème année. Ce cours de 30h (20h Vol.1, 10h Vol.2) couvrirait les bases de l'IA et serait coordonné par le Pr. Dubois."

---

**Note** : Ce système est conçu pour faciliter la gestion des demandes de modification tout en maintenant la traçabilité et la cohérence des données. Toute demande sera examinée par l'administration avant traitement.
